3. NodeJS :-

1. Node.js is a server side platform built on Google Chrome's JavaScript Engine (V8 Engine).

2. Node.js is an open source, cross-platform runtime environment for developing server-side and networking applications.

3. Node.js applications are written in JavaScript, and can be run within the Node.js runtime on OS X, Microsoft Windows, and Linux.
